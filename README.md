# 마니또 뽑기 웹앱 🎁

GitHub Pages를 통해 배포되는 마니또 뽑기 웹 애플리케이션입니다.

## 기능

- ✨ 참가자 이름을 입력하여 마니또를 뽑습니다
- 🎯 각 사람마다 서로 다른 마니또가 배정됩니다 (겹치지 않음)
- 🔒 결과는 기본적으로 가려져 있어, 클릭해야만 확인할 수 있습니다
- 👆 각 사람의 마니또를 클릭하면 나타나고, 다시 클릭하면 가려집니다
- 📱 반응형 디자인으로 모바일과 데스크톱에서 모두 사용 가능합니다

## 사용 방법

1. 이름 입력 칸에 참가자들의 이름을 한 줄에 하나씩 입력합니다
2. "마니또 뽑기" 버튼을 클릭합니다
3. 결과에서 각 사람의 이름을 클릭하여 자신의 마니또를 확인합니다
4. 다시 클릭하면 결과가 가려집니다

## 배포 방법

1. 이 저장소를 GitHub에 업로드합니다
2. 저장소 설정(Settings) → Pages로 이동합니다
3. Source에서 배포할 브랜치(보통 `main` 또는 `master`)를 선택합니다
4. 저장 후 몇 분 후 `https://[사용자명].github.io/[저장소명]` 또는 `https://[사용자명].github.io` (저장소 이름이 `[사용자명].github.io`인 경우)로 접속할 수 있습니다
